CardioBrain

A freely available software for assessing dynamic cerebral autoregulation via Transfer Function Analysis (TFA) on continuously recorded blood pressure (ABP) and cerebral blood flow (CBF) signals.

About

Cerebral autoregulation (CA) is a critical mechanism maintaining cerebral blood flow relatively stable despite arterial blood pressure fluctuations. CardioBrain performs spectral density and cross-spectral analysis via the Welch method, computing gain, phase, and coherence across frequency bands.

The software includes tools to address signal misalignments (temporal shift), and enforces adherence to recommended standards for TFA analysis. No programming skills required.
